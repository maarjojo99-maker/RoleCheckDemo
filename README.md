# Role Check Demo

An authentication and role-based access control (RBAC) demonstration in React.

## Features
- Mock login system for different user roles (Admin vs. Letter Approval Authority).
- Persistent session storage using cookies.
- Conditional UI rendering based on user permissions.
- Role-based dashboard views.

## Technical Details
- Custom cookie management utilities.
- State-based role switching for demonstration purposes.

## Getting Started
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev`.
